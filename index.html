Projects

Flames

The FLAMES game in Python is a fun program that predicts relationships between two names by comparing and eliminating common letters. The remaining letters are counted and used to cyclically eliminate letters from the acronym **FLAMES** (Friends, Lovers, Affectionate, Marriage, Enemies, Siblings). The last remaining letter determines the relationship type. Python simplifies the process by handling string manipulations, counting, and looping, making it an engaging project to practice basic programming concepts like loops, conditionals, and string operations.

Criminal DBMS

A Criminal Database Management System (DBMS) using Java Swing and phpMyAdmin is a desktop application designed to manage criminal records efficiently. The user interface, built with Java Swing, allows users to perform operations like adding, updating, searching, and deleting records. The backend database, managed with **MySQL** through **phpMyAdmin**, securely stores criminal data such as personal details, crime history, and case status. Java connects to the database via **JDBC**, enabling seamless communication between the application and the database. This project is ideal for learning database integration, GUI design, and CRUD operations, making it useful for law enforcement agencies to track and manage records effectively.

AI-powered Traffic light control system

An AI-powered Traffic Light Control System using React.js, Flask, and the YOLOv4-tiny pre-trained model is designed to optimize traffic flow through real-time vehicle detection. The frontend, developed with React.js, provides an interactive interface for visualizing live traffic data. The backend, powered by Flask, handles communication between the frontend and the YOLOv4-tiny model, which processes live video feeds to detect and classify vehicles efficiently. The system dynamically adjusts traffic light durations based on traffic density, improving efficiency and reducing congestion. This project combines modern web development, machine learning, and computer vision to deliver a smart traffic management solution.
